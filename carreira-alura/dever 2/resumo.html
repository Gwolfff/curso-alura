Dever 2

A aula foi falado sobre direitos autorais de imagens que estão na rede, que tem quer ter cuidado quando for usar no seu projeto por que ela pode dar copyright no seu projeto.

E foi falado também sobre a dimensão das imagens que são usadas no sites que pode deixar o site pesado, e quando for avaliado pelo motor de busca do google o site pode ser mau avaliado e ficar em uma posição ruim.

Ao falar de sites, consideramos que pessoas acessarão por tablets e celulares, então precisamos de imagens nessas dimensões também. O processo é o mesmo: em "Image", selecionamos "Scale Image" e ajustamos. Para tablets, a largura máxima é 720 pixels. Mantemos a proporção, ajustamos a qualidade para 70 e salvamos como paisagem_720. Para celulares, a largura máxima é 480 pixels. Salvamos como paisagem_480, mantendo a qualidade em 70.

Tabela com as principais tags

Tag Descrição

<html> Define o documento como HTML.
<head> Contém metadados sobre o documento (ex.: título, links de estilo).
<title> Define o título da página exibido na aba do navegador.
<body> Contém todo o conteúdo visível da página (textos, imagens, links, etc.).
<h1> a <h6> Títulos e subtítulos, do mais importante (h1) ao menos importante (h6).
<p> Define um parágrafo de texto.
<a> Cria links para outras páginas ou recursos.
<img> Insere imagens no documento.
<ul> Cria uma lista não ordenada (com marcadores).
<ol> Cria uma lista ordenada (numerada).
<li> Define um item dentro de listas (ul ou ol).
<div> Define uma divisão ou seção genérica do documento.
<span> Marca um trecho de texto inline para aplicar estilo ou identificação.
<table> Cria uma tabela.
<tr> Linha dentro de uma tabela.
<td> Célula de uma tabela.
<form> Define um formulário para entrada de dados do usuário.
<input> Campo de entrada de dados dentro de um formulário.
<button> Cria um botão clicável.
